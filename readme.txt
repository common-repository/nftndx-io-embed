Adds support for embedding NFTs minted on the blockchain and tracked on NFTNDX.IO in Block Editor (Gutenberg) and Classic Editor.

== Description ==

Adds support for embedding NFTs minted on the different blockchain networks like Ethereum in Block Editor (Gutenberg) and Classic Editor by providing a card displaying all the necessary information.

== Usage ==

To use the plugin and embed the NFTs cards tracked on NFTNDX.IO in your posts, you can either use the visual editor by selecting the embed block and setting the URL of the token, or you can use the shortcode below.

**Example:**

"EVERYDAYS: THE FIRST 5000 DAYS" NFT by Beeple

(The NFT was sold for $69 Millions)

`
[embed]https://nftndx.io/token/0x2A46f2fFD99e19a89476E2f62270e0a35bBf0756-40913[/embed]
`

== Frequently Asked Questions ==

= I can't find the NFT I wanted to embed in my post on NFTNDX.IO, What should I do? =

NFTNDX.IO is always growing and trying to provide a great overview of the NFT ecosystem. If you can't find the NFT you wanted to list, feel free to send us an e-mail at nft@nftndx.io or you can fill this form and submit it https://nftndx.io/submit.

= This version requires PHP 7.1 =

Yes, this plugin requires PHP 7.1 or later. That is by design.
